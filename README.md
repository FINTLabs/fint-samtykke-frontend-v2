# FINT Samtykke v2

Se, gi og trekk samtykke.

## Getting Started

### Installation

Install the dependencies:

```bash
npm install
```

### Development

Start the development mockServer with HMR:

```bash
npm run dev:mock
```

Your application will be available at `http://localhost:8000`.

## Building for Production

Create a production build:

```bash
npm run build
```
